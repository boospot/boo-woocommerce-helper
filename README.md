# boo-woocommerce-helper
Another WooCommerce Helper


## Composer

composer.json file: 

```json
{
  "require": {
    "boospot/boo-woocommerce-helper": "^1.0"
  }
}
```

or using command line:

```shell script
composer require "boospot/boo-woocommerce-helper"
```
